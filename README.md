# Mumbai University CGPA Calculator 🎓

A web-based CGPA calculator built using HTML, CSS, and JavaScript.

## Features
- Percentage ↔ CGPA Conversion
- Subject-wise Marks Calculation
- Credit-Based CGPA Calculation
- KT Detection
- Clean Responsive UI
